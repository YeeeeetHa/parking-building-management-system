# Smart Parking Lot Management System

A Java-based web application designed to manage parking building facilities, track real-time slot occupancy, process check-ins and check-outs, automate fee calculations, and support advance slot reservations. 

---

## 🛠️ Technology Stack
* **Backend:** Java Servlets, JDBC, MS SQL Server
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Build Tool:** Apache Ant
* **Web Server:** Apache Tomcat
